# Smart Government – AI Agentic Hackathon
[🇬🇧 English version](./eng/README.md)

Welkom bij de Smart Government AI Agentic Hackathon. Deze repository bevat de handleidingen en labs voor de hackathon, gericht op het bouwen van slimme AI-agenten voor de overheid.

## Samenvatting van de Labs

De hackathon is opgedeeld in twee delen, elk met een eigen handleiding:

### [Deel 1: Data Agents in Fabric](./Data_Agent_Labs.md)
In dit deel ga je aan de slag met **Microsoft Fabric Data Agents**. Je leert hoe je deze slimme assistenten kunt inzetten om gemakkelijker met data te werken via natuurlijke taal.

**Onderwerpen:**
*   Opzetten van de Microsoft Fabric omgeving.
*   Configureren van een Data Agent.
*   Aan de slag met het bevragen van data in een Lakehouse.

### [Deel 2: Agent Lab](./Agent_Labs.md)
Dit deel richt zich op het ontwikkelen van **AI Agents** met behulp van **Azure AI Foundry**. Je leert hoe LLM's, context (RAG) en tools samenkomen om taken zelfstandig uit te voeren.

**Onderwerpen:**
*   Inloggen en verkennen van Microsoft Azure AI Foundry.
*   Aanmaken van je eerste Agent.
*   Koppelen van tools zoals Fabric Data Agent en Foundry IQ.
*   Begrijpen van agent workflows en het Model Context Protocol (MCP).

Veel succes met de hackathon!
