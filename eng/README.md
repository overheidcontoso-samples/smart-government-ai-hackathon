# Smart Government – AI Agentic Hackathon

Welcome to the Smart Government AI Agentic Hackathon. This repository contains the manuals and labs for the hackathon, focused on building smart AI agents for the government.

## Summary of the Labs

The hackathon is divided into two parts, each with its own manual:

### [Part 1: Data Agents in Fabric](./Data_Agent_Labs.md)
In this part you will get started with **Microsoft Fabric Data Agents**. You will learn how to use these smart assistants to work with data more easily via natural language.

**Topics:**
*   Setting up the Microsoft Fabric environment.
*   Configuring a Data Agent.
*   Asking questions to data in a Lakehouse.

### [Part 2: Agent Lab](./Agent_Labs.md)
This part focuses on developing **AI Agents** using **Azure AI Foundry**. You will learn how LLMs, context (RAG) and tools come together to perform tasks independently.

**Topics:**
*   Logging in and exploring Microsoft Azure AI Foundry.
*   Creating your first Agent.
*   Connecting tools such as Fabric Data Agent and Foundry IQ.
*   Understanding agent workflows and the Model Context Protocol (MCP).

Good luck with the hackathon!
